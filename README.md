# harisn.de

> Eine moderne, datenschutzorientierte Portfolio-Website, erstellt mit semantischem HTML, CSS und Vanilla JavaScript.
> A modern, privacy-focused portfolio website built with semantic HTML, CSS, and Vanilla JavaScript.

## Live Site / Live-Webseite
- **Main Site / Hauptseite:** [https://harisn.de](https://harisn.de)
- **GitHub Pages:** [https://harisnae.github.io/harisn.de](https://harisnae.github.io/harisn.de)

## Tech Stack / Technologie-Stack
- **HTML5** (Semantic Markup / Semantisches Markup)
- **CSS3** (Modern Styling, Flexbox/Grid)
- **JavaScript (Vanilla ES6+)
- **Hosting / Hosting:** AWS S3 + CloudFront & GitHub Pages

## Features / Funktionen
- Fully responsive design (Mobile & Desktop / Mobil & Desktop)
- Dark mode support (System preference / Systemeinstellung)
- No external frameworks (Lightning fast load times / Blitzschnelle Ladezeiten)
- SEO optimized structure (SEO optimierte Struktur)

## Setup / Einrichtung
1. Clone the repository: `git clone https://github.com/harisnae/harisn.de.git`
2. Open `index.html` in your browser / Öffnen Sie `index.html` in Ihrem Browser.
